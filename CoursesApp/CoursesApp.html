Courses CESAE Digital

A simple Android app for managing the CESAE Digital courses.

Developed during the Software Developer Course by CESAE Digital (2023).



Features:

  • Created an Android application in Android Studio with Kotlin
  • Splash screen and App Icon
  • Implemented a login system with three users and displays a Toast message for login validation.
  • Displays the course details: name, location, start date, end date, price, duration, edition.
  • Allows users to insert, edit, and delete courses.
  • Allows sorting of courses by name and start date (ascending and descending).
  • The application is multilingual, supporting both Portuguese and English.
